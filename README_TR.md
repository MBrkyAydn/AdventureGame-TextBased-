# Adventure Game

Bu proje, Java OOP mantığını öğrenmek ve pekiştirmek için geliştirilmiş konsol tabanlı bir macera oyunudur. Oyuncu farklı lokasyonlara giderek canavarlarla savaşır, para kazanır, mağazadan silah ve zırh satın alır ve envanterine özel eşyalar ekler.

## Kullanılan Teknolojiler

* Java
* OOP
* Inheritance
* Abstract Class
* Encapsulation
* Constructor
* Array / Random
* Console Input

## Oyundaki Temel Özellikler

* Karakter seçimi
* Farklı lokasyonlara gitme
* Canavarlarla savaşma
* Mağazadan silah ve zırh satın alma
* Envanter sistemi
* Para, can ve hasar yönetimi
* Güvenli evde can yenileme
* Bölgelere özel ödül/loot sistemi

## Lokasyonlar

* Safe House
* Mağara
* Orman
* Nehir
* Mağaza

## Amaç

Oyuncunun amacı farklı bölgelerdeki düşmanları yenerek özel eşyaları toplamak ve oyunu tamamlamaktır.

## Bu Projede Öğrendiklerim

Bu proje sayesinde Java'da sınıf yapısı, nesne oluşturma, inheritance, abstract class kullanımı, constructor mantığı, method kullanımı ve nesneler arası veri aktarımı gibi temel OOP konularını pratik ettim.

## Çalıştırma

Projeyi çalıştırmak için Java kurulu olmalıdır. Main sınıfı çalıştırılarak oyun başlatılabilir.

```bash
javac Main.java
java Main
```
