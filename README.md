# Adventure Game

This project is a console-based adventure game developed in Java to practice and improve Object-Oriented Programming concepts. The player can travel to different locations, fight monsters, earn money, buy weapons and armor, and collect special items.

## Technologies Used

* Java
* Object-Oriented Programming
* Inheritance
* Abstract Classes
* Encapsulation
* Constructors
* Arrays
* Random class
* Console input

## Features

* Character selection
* Different game locations
* Monster combat system
* Weapon and armor shop
* Inventory system
* Health, damage and money management
* Health recovery in the Safe House
* Location-based reward and loot system

## Locations

* Safe House
* Cave
* Forest
* River
* Tool Store

## Game Objective

The main goal of the game is to defeat enemies in different locations, collect special items, and complete the adventure successfully.

## What I Learned

In this project, I practiced core Java OOP concepts such as classes, objects, inheritance, abstract classes, constructors, method usage, encapsulation, and passing objects between classes.

## How to Run

Make sure Java is installed on your computer. Then run the `Main` class to start the game.

```bash
javac Main.java
java Main
```
